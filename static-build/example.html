BetterFaster Media Static Website

This is a demonstration page showing how to embed the BetterFaster Media static website into another HTML page.

Integration Options

Option 1: Full-Page Integration

To serve the static website directly, simply host the index.html file and its associated assets.

View the Full Website

Option 2: Iframe Embedding

You can embed the website in an iframe on another page:

<iframe src="./index.html" width="100%" height="600px" frameborder="0"></iframe>
Note: The BetterFaster Media website is designed as a standalone site and works best when served directly rather than embedded in an iframe.

File Structure

The static website consists of:

  • index.html - Main HTML file
  • assets/ - Directory containing JavaScript and CSS files
